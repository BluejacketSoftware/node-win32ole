NAME

node-win32ole - Asynchronous, non-blocking win32ole bindings for node.js .

USAGE

Install with npm install node-win32ole.

var win32ole = require('win32ole');
var xla = new win32ole.Dispatch('Excel.Application');

xla.series([
  function(){
    var wb = xla.Workbooks.Open('test/tmp/test.xls');
  },
  function(){
    var book = wb.Books(1);
  },
  function(){
    var ws = book.WorkSheets(1);
  },
  function(){
    book.Save();
    book.Close();
    xla.Quit();
  }
]);

BUILDING

This project uses VC++ 2008 Express (or later) and Python 2.6 (or later) .
(When using Python 2.5, it needs multiprocessing 2.5 back port .)

Bulding also requires node-gyp to be installed. You can do this with npm:

npm install -g node-gyp

To obtain and build the bindings:

git clone git://github.com/idobatter/node-win32ole.git
cd node-win32ole
node-gyp configure
node-gyp build

You can also use npm to download and install them:

npm install node-win32ole

TESTS

mocha is required to run unit tests.

npm install -g mocha
nmake /a test
